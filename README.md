# Herryanto's dotfiles

## Setup

```sh
git clone https://github.com/jugend/dotfiles.git
cd dotfiles
source setup.sh
source brew.sh
```

## Local files

Local files supported:
- `aliases.local`
- `bash_profile.local`
- `tmux.conf.local`
- `vimrc.local`
